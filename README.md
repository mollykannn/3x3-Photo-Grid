# 3x3-Photo-Grid

Create your 3x3 Photo Grid.

[https://mollykannn.github.io/3x3-Photo-Grid](https://mollykannn.github.io/3x3-Photo-Grid)
![Screenshot](https://mollykannn.github.io/3x3-Photo-Grid/img/screenshot.png)


## 安裝 (Install)

```shell
pnpm install
```

## 使用方法 (Usage)

建立檔案 (Create files)
```shell
pnpm run build
```

運行 (Run)
```shell
pnpm run dev
```
